Short & Simple Explanation

1. Control Unit (CU)

The Control Unit directs all operations of the CPU. It tells other parts what to do and manages instructions.

2. Arithmetic Logic Unit (ALU)

The ALU performs calculations like addition, subtraction, AND, OR, and comparisons.

3. Registers

Registers are small, fast memory locations inside the CPU that store temporary data.

4. Cache Memory

Cache is high-speed memory inside the CPU that stores frequently used data for faster access.
